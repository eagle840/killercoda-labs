# Welcome to Basic AWS Monitoring & Governance on MiniStack

In this lab, you will explore the essential tools for monitoring and governing your AWS environment using **MiniStack**.

Observability and governance are critical for maintaining a secure and cost-efficient cloud environment. You will learn how to organize resources with tags, monitor system health with CloudWatch, and audit API activity with CloudTrail.

### What you will build:
*   **Tags** to manage and track resource ownership.
*   **CloudWatch** dashboards and logs for observability.
*   **CloudTrail** to audit API activity and configuration changes.

Let's get started by setting up your local environment!
