# Welcome to Basic AWS Storage on MiniStack

In this lab, you will explore the various storage services available in AWS using **MiniStack**.

Effective data management is crucial for any cloud application. You will gain hands-on experience with Object storage (S3), NoSQL databases (DynamoDB), and Relational databases (RDS), covering different persistence strategies.

### What you will build:
*   **S3** buckets for object storage.
*   **DynamoDB** tables for NoSQL data.
*   **RDS** instances for relational database management.

Let's get started by setting up your local environment!
