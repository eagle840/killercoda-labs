# Welcome to Basic AWS Serverless on MiniStack

In this lab, you will explore event-driven architecture using **MiniStack**.

Serverless compute and messaging services are the foundation of modern, scalable applications. You will learn how to trigger code execution based on events and decouple your services using messaging queues and event buses.

### What you will build:
*   **Lambda** functions for serverless compute.
*   **SQS** queues and **SNS** topics for asynchronous messaging.
*   **EventBridge** for event-driven routing.

Let's get started by setting up your local environment!
