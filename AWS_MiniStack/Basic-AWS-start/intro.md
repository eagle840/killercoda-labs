# Welcome to Basic AWS Identity & Access (IAM) on MiniStack

In this lab, you will explore the foundational security layer of AWS using **MiniStack**.

IAM is the cornerstone of AWS security, defining **who** (Users/Roles) can do **what** (Policies) on **which** resources. Mastering these concepts is essential before building any other cloud infrastructure.

### What you will build:
*   **IAM Users** and managed access policies.
*   **IAM Roles** for secure, temporary credential delegation.
*   **Governance guardrails** using Service Control Policies (SCPs).
