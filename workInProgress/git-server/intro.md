# Introduction

In this lab, you will deploy a self-hosted Git server using **Gitea**.

Gitea is a lightweight, self-hosted Git service that is easy to install and run. By the end of this lab, you will have a functional Git server, configured for SSH access, ready for you to create and manage repositories.

**Learning Objectives:**
1. Deploy Gitea using Docker Compose.
2. Perform initial Gitea UI configuration.
3. Set up SSH key authentication for secure repository access.
4. Create a repository and push code to your new server.
