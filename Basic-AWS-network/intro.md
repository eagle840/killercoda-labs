# Welcome to Basic AWS Networking on MiniStack

In this lab, you will learn how to design and implement a virtual network environment on AWS using **MiniStack**.

Networking is the foundation of any cloud architecture. You will gain hands-on experience with VPCs, Subnets, and Security Groups, essential components for securing and connecting your cloud resources.

### What you will build:
*   A custom Virtual Private Cloud (**VPC**).
*   Isolated **Subnets**.
*   **Internet Gateways** for connectivity.
*   **Security Groups** to control traffic flow.

Let's get started by setting up your local environment!
