# Lab Plan: Agentic AI with Ollama and AWS MiniStack

1. [ ] **Setup Infrastructure**: Configure and verify the AWS MiniStack container and the `lab_net` network.
2. [ ] **Setup Inference Engine**: Install Ollama and pull the lightweight `qwen2.5:0.5b` model.
3. [ ] **Prepare Agent Code**: Review and deploy the provided Python agent script that integrates `boto3` (AWS) and `ollama` (LLM).
4. [ ] **Orchestrate Agent**: Run the agent to monitor an S3 bucket and trigger automated logic based on LLM analysis.
5. [ ] **Verify & Finalize**: Confirm the agent successfully processed the S3 event and generated the expected output, then clean up the environment.
