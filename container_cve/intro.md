# Software Composition Analysis (SCA) & Container CVE Scanning with Trivy

Welcome! In this hands-on lab, you will learn how to use **Trivy** (by Aqua Security)—a fast, comprehensive vulnerability scanner for containers, code dependencies, and configuration files.

## 🎯 What You Will Learn
1. **Container Image Scanning:** Install Trivy and run checks against base container images to find CVEs.
2. **Visual Reporting:** Export scans as beautiful HTML reports and view them in a local web dashboard.
3. **Software Composition Analysis (SCA):** Detect vulnerabilities inside language-specific package dependencies (like `package.json`).
4. **Configuration Auditing:** Scan `Dockerfile` settings for security misconfigurations.
5. **CI/CD Integration:** Gate builds by failing pipelines on critical CVEs, and bypass specific risks using a `.trivyignore` file.
6. **Hands-on Remediation:** Fix the vulnerabilities and configurations, then verify that the security checks pass successfully!
